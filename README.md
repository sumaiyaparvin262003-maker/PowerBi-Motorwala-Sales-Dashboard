
 Motorola Sales Dashboard

 Project Overview
This project is an interactive Power BI dashboard developed to analyze Motorola sales performance across different cities, brands, mobile models, payment methods, and time periods. The dashboard provides key business insights to support data-driven decision-making.

 Tools Used
- Power BI
- Microsoft Excel

 Key Performance Indicators (KPIs)
- Total Sales: 769M
- Total Quantity Sold: 19K
- Total Transactions: 4K
- Average Price: 40K

 Dashboard Features
- Sales analysis by city using map visualization
- Monthly quantity trend analysis
- Customer ratings analysis
- Payment method distribution
- Brand-wise sales and transaction analysis
- Mobile model performance comparison
- Sales analysis by day of the week
- Interactive slicers for:
  - Month
  - Mobile Model
  - Payment Method
  - Brand
  - Day Name

 Key Insights
- Sales performance can be monitored across multiple cities in India.
- UPI, Debit Card, Credit Card, and Cash payment methods contribute significantly to transactions.
- Different mobile brands show varying sales and transaction volumes.
- Monthly sales quantity trends help identify peak and low-performing periods.
- Customer ratings provide insight into customer satisfaction levels.
